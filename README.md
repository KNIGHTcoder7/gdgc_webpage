This repository hosts the GDG OnCampus MLRIT website built with React, Vite, TypeScript and Tailwind CSS.

Getting started

```sh
git clone <YOUR_REPO_URL>
cd <YOUR_PROJECT_DIR>
npm install
npm run dev
```

Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS

Deployment

Build for production:

```sh
npm run build
```

Then serve the `dist` output with your preferred static hosting.

If you need help customizing the site or removing additional third-party references, open an issue or ask here.
